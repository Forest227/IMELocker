# 输入法锁定 v0.0.1

首个公开版本，提供一个尽量轻量的 macOS 菜单栏工具，用来把系统输入法稳定锁定到指定目标输入法。

## 新增

- 菜单栏常驻运行，可快速开启或关闭输入法锁定
- 可从菜单中选择目标输入法
- 监听系统输入法切换，并自动切回目标输入法
- 增加周期性检查，降低输入法被其他应用切走后无法自动恢复的情况
- 支持登录时启动
- 支持从菜单中重启应用

## 改进

- 应用名称统一为“输入法锁定”
- 更新应用图标与打包资源
- 增加 `.dmg` 安装包，方便直接分发

## 使用提示

- 建议将应用拖到 `Applications` 后再开启“登录时启动”
- 遇到 `Secure Input` 场景时，应用会暂停强制切换
- 当前版本未做开发者签名和公证，首次打开可能需要手动放行
