# 输入法锁定 v0.0.2

## 新增

- 点击输入法列表项直接锁定，再次点击同一项取消锁定
- 新增「关于」菜单项，显示版本信息与项目链接
- 状态栏图标改用盾牌锁样式（lock.shield），更符合 macOS 安全语义

## 改进

- **UI 重新设计**：粗体状态标题、分组布局、所有菜单项带 SF Symbol 图标
- **SF Symbol 兼容性**：每个图标维护降级回退链，确保 macOS 12 所有版本正常显示
- **线程安全**：NSLock 替换为串行 DispatchQueue，避免潜在死锁
- **输入法列表缓存**：减少重复调用 TISCreateInputSourceList
- **Timer 生命周期**：锁定禁用时停止周期性检查，减少无用 CPU 唤醒
- **同步执行**：用户主动操作时直接执行 enforce，跳过防抖延迟
- **delegate 生命周期**：main.swift 使用 withExtendedLifetime 防止提前释放
- **构建优化**：arm64 与 x86_64 并行编译

## 移除

- 移除独立的「锁定启用」复选框，改为点击列表项切换
- 移除冗余的 `setActivationPolicy` 调用
