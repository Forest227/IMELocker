# 输入法锁定

`输入法锁定` 是一个极简的 macOS 菜单栏工具，用来把系统输入法稳定锁定到你选定的目标输入法。

当某些应用把输入法切到 `ABC` 或其他输入源时，应用会自动尝试切回你指定的输入法。除了监听系统输入法切换通知，它还会做周期性检查，减少“突然被切走且回不来”的情况。

## 功能

- 锁定到指定输入法，例如微信输入法
- 菜单栏快速开关锁定状态
- 自动检测并切回目标输入法
- 支持登录时启动
- 检测到 `Secure Input` 时自动暂停强制切换，避免密码框场景反复抢输入法
- 提供重启菜单项，方便刷新运行状态

## 系统要求

- macOS 12 或更高版本
- 建议 macOS 13+ 以获得更稳定的“登录时启动”体验

## 下载

预编译版本会发布在 GitHub Releases：

[Releases](https://github.com/Forest227/IMELocker/releases)

下载 `.dmg` 后，把 `输入法锁定.app` 拖到 `Applications` 即可。

注意：

- 当前发布版本未做开发者签名和 Apple 公证，首次打开时 macOS 可能会提示安全警告
- 如果被系统拦截，可以在“系统设置 → 隐私与安全性”中选择仍要打开

## 使用

启动应用后，它会出现在菜单栏中。

首次使用建议：

1. 打开菜单栏图标。
2. 确认 `锁定启用` 已打开。
3. 在 `目标输入法` 里选择你想锁定的输入法。
4. 如果需要开机自动运行，再打开 `登录时启动`。

## 已知限制

- macOS 在密码框等 `Secure Input` 场景下可能禁止第三方输入法工作；本工具会在这类场景暂停强制切换
- 某些应用会主动调用系统 API 改写输入法，本工具只能在切换发生后再尝试切回
- “登录时启动”依赖系统登录项能力，旧系统上可能不可用或需要用户额外批准

## 从源码构建

在仓库根目录执行：

```bash
./build.sh
```

生成产物：

```bash
build/输入法锁定.app
```

本地打包 `.dmg`：

```bash
./scripts/build_dmg.sh
```

默认生成：

```bash
build/IMELocker-v0.0.1.dmg
```
