# 输入法锁定

`输入法锁定`（`IMELocker`）是一个轻量的 macOS 菜单栏工具，用来把系统输入法稳定锁定到你指定的目标输入法。

它适合这种场景：你明明已经切到微信输入法、搜狗或别的中文输入法，但某些应用会突然把输入法切回 `ABC`、英文或别的输入源，而且还不容易切回来。这个工具会在检测到切换后，自动把输入法拉回你选定的目标输入法。

除了监听系统输入法变化通知，它还会做周期性检查，所以在通知没有及时触发的情况下，也能兜底恢复。

## 功能特性

- **点击即锁定** — 在菜单中点击目标输入法即锁定，再次点击同一项取消锁定
- **自动恢复** — 监听系统输入法切换、前台应用切换，加上周期性检查，三重保障
- **Secure Input 检测** — 密码框等安全输入场景自动暂停强制切换
- **登录时启动** — 支持 macOS 13+ 的 SMAppService 登录项
- **通用二进制** — 同时支持 Apple Silicon 和 Intel Mac
- **SF Symbol 图标** — 菜单栏与菜单项均有图标，macOS 12 全版本兼容

## 系统要求

- macOS 12 或更高版本
- 建议 macOS 13+ 以获得更稳定的「登录时启动」体验

## 下载与安装

发布版本在 GitHub Releases：

[Releases](https://github.com/Forest227/IMELocker/releases)

安装步骤：

1. 下载最新的 `.dmg` 安装包。
2. 打开后把 `输入法锁定.app` 拖到 `Applications`。
3. 首次启动时如果被系统拦截，在「系统设置 → 隐私与安全性」中选择仍要打开。

> **注意：** 当前发布版本未做开发者签名和 Apple 公证，首次打开可能会看到 macOS 安全提示。建议把应用放到 `Applications` 后再开启「登录时启动」。

## 使用方法

启动后，应用会以锁形图标出现在菜单栏中。

### 锁定到目标输入法

1. 点击菜单栏锁图标打开菜单。
2. 在「选择目标输入法」中点击你想锁定的输入法，即完成锁定。
3. 再次点击同一输入法即可取消锁定。

### 菜单布局

```
🛡 输入法已锁定                    ← 状态标题
当前：微信输入法  ▸ Secure Input   ← 当前输入法 + 安全输入状态
───
选择目标输入法：                   ← 点击锁定，再点解锁
✓ 微信输入法
  ABC
  简体拼音
───
⏻ 开机自启                        ← 登录时启动开关
───
↻ 重启应用
ℹ 关于「输入法锁定」
───
✕ 退出                           ⌘Q
```

### 状态栏图标

- 🛡 **盾牌锁（绿色）** — 锁定已启用
- 🛡 **盾牌锁（灰色）** — 锁定已关闭

## 工作原理

应用通过 macOS Carbon 框架的 TIS（Text Input Source）API 获取和切换输入法，在以下时机尝试恢复目标输入法：

1. **系统通知** — 监听 `kTISNotifySelectedKeyboardInputSourceChanged`，输入法被切换时立即响应
2. **应用切换** — 监听 `NSWorkspace.didActivateApplicationNotification`，前台应用变化时检查
3. **周期性检查** — 每 5 秒检查一次，兜底防止通知遗漏

如果当前处于 `Secure Input` 场景（密码输入框、系统授权窗口等），应用会暂停强制切换，避免反复抢输入法。

## 已知限制

- macOS 在密码框等 `Secure Input` 场景下可能禁止第三方输入法工作；本工具会在这类场景暂停强制切换
- 某些应用会主动调用系统 API 改写输入法，本工具只能在切换发生后再尝试切回
- 「登录时启动」依赖系统登录项能力，macOS 12 上不可用，macOS 13+ 需要用户批准
- 当前仓库提供的是未签名版本，企业环境或更严格的安全策略下可能需要手动放行

## 从源码构建

在仓库根目录执行：

```bash
./build.sh
```

生成产物：

```
build/输入法锁定.app
```

构建脚本会自动编译 arm64 和 x86_64 并合并为通用二进制。如果 x86_64 编译失败，会降级为仅 arm64。

## 打包 DMG

本地生成安装包：

```bash
./scripts/build_dmg.sh
```

默认输出：

```
build/IMELocker-v0.0.2.dmg
```

也可以指定版本号：

```bash
./scripts/build_dmg.sh 0.0.2
```
